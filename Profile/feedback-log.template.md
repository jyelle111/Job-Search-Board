# Feedback Log

_Every approve/pass/edit decision gets logged here with the "why," so matching and
drafting improve over time. Newest first._

_Copy this file to `feedback-log.md` (drop the `.template`) to start — it begins empty.
The scheduled tasks and skills in this system append to it automatically (passed jobs with
reasons, answered daily questions, comp figures stated to specific companies); you can also
add your own entries any time._
