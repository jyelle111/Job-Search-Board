# Context Drop Folder

Drop any files that give Claude context about a role or company here — interview
transcripts, recruiter emails, JD PDFs, notes, comp screenshots.

How to organize: create a subfolder named with the job id from the board (e.g.
`2026-07-06-example-corp-senior-role/`) or just the company name. Anything here gets read
during check-ins and used for resume tailoring, interview prep, and follow-ups.

You can also paste transcripts directly into a job's note box on the board — long text is
fine.

Real content dropped here is gitignored (only this README is tracked) — this is your data,
not something to commit to a shared repo.
